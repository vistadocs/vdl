The Clinical Capacity Search Tool (CCST) is a new web tool utilizing the JavaScript React framework and provides users with a single user interface for scheduling staff to identify, in real-time, the best-fit available appointments for Veteran patients that align with the clinical services and clinical encounter modalities needed to provide health care. The purpose of CCST is to provide a single point of access to available appointment data, which reduces operating costs and improves operational efficiency.
