<p><strong>Benefits of TIU</strong></p>
</blockquote>
<ul>
<li><p>Ability to view and manipulate clinical documents from a single program</p></li>
<li><p>Upload capability for many types of documents</p></li>
<li><p>Improved searching and retrieving capabilities across documents</p></li>
<li><p>Less duplication of effort for clinicians, managers, and other staff</p></li>
<li><p>Boilerplate support</p></li>
<li><p>Links to the Computerized Patient Record System (CPRS), Authorization/ Subscription Utility (ASU), Incomplete Record Tracking (IRT), and interfacility document transfer</p></li>

<p><em><strong>How to enter a new Progress Note:</strong></em></p>
<p><strong>1.</strong> Type in the patient name.</p>
<p><strong>2.</strong> Type in a Progress Note Title.</p>
<p><strong>3.</strong> If prompted, select an existing Visit or create a new Visit. (Every outpatient episode of care must now be associated with a Visit or Encounter—most will be entered automatically by the admission or checkout process).</p>
<p><strong>4.</strong> Type in the text of the Note. If it’s a SOAP Note or there's a boilerplate for this, you can fill in the blanks or edit existing text. You can use the FileMan text editor or full-screen editor.</p>
<p><strong>5.</strong> Enter your electronic signature.</p>
<p><strong>6.</strong> If you wish, print the note now.</p>
