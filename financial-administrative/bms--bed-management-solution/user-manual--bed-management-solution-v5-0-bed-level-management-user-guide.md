## Purpose

<!-- back-to-toc -->[↑ Table of Contents](#table-of-contents)

For users who need to categorize and track beds according to unit/ward-based Bed Levels (commonly referred to as “bed types”), this document has been designed so that those users can operate the Bed Management Solution (BMS) application effectively.

## Intended Audience

<!-- back-to-toc -->[↑ Table of Contents](#table-of-contents)

This guide provides instructions on how to use the BMS Bed Level Management tool. Unlike the legacy BMS Bed Type Update Tool (“BED TYPE CROSSWALK”), Bed Level Management allows for *bed-specific* configurations, including the ability to apply a Bed Level value in multiple ways—whether as a *primary* level, a *current* level or one or more *available* levels.

The target audience for this manual will be administrative staff (that is, those with the BMS Admin User role) working in Veterans Affairs (VA) facilities or Veterans Integrated Services Networks (VISNs). This user guide assumes the identified users will have basic computer software knowledge and some previous experience using BMS in the healthcare setting.

![](bed-management-solution-version-5-0-bed-level-management-user-guide/002.png)For additional information regarding comprehensive use of BMS for your facility or VISN, please refer to the *Bed Management Solution (BMS) User Guide*, the *Bed Management Solution (BMS) Admin Guide*, the *Bed Management Solution (BMS) Community Care Tracking List User Guide*, or the *Bed Management Solution (BMS) Technical Manual*.

# Getting Started

<!-- back-to-toc -->[↑ Table of Contents](#table-of-contents)

Bed Management Solution (BMS) now includes a bed type update tool within the BMS application itself. This *Bed Level Management* screen uses facility-based Ward Group searching to display individual beds editable one-by-one or in batch.

<span id="_Toc205212182" class="anchor"></span>Figure 2 - Bed Level Management - Main Screen

![](bed-management-solution-version-5-0-bed-level-management-user-guide/008.png)

![](bed-management-solution-version-5-0-bed-level-management-user-guide/009.png) Additional reporting can be found on the [BMS Reports](https://dvagov.sharepoint.com/sites/VACOVHADUSHOM/HOC/BMS/review/SitePages/home_buttons.aspx) page.
